## Description

This p5.js project is a simple caf√© ordering simulator. The player makes two choices, drink temperature (hot or cold) and taste preference (sweet or bitter). Selecting these choices involves using on-screen buttons. Based on these selections, the code generates a final drink order. The project demonstrates basic screen navigation, user input handling, and simple UI design.

## Learning Goals

- Practice managing multiple screens using a shared game state
- Implement clickable buttons for user interaction
- Track user choices across multiple steps
- Display a generated result based on user input
